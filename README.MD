Simple  Minecraft-UserData
======================
## Introduction
### A Simple Mojang api based minecraft module
## Installation
```bash
npm install minecraft-userdata
```
## Usage
```js
const mc = require("minecraft-userdata")
mc.GetData("<username>").then(user => console.log(user.id))
```
